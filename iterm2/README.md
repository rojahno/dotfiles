# iTerm2 Configuration

## Backup Preferences

To backup your iTerm2 settings:

1. Open iTerm2 → Settings → General → Preferences
2. Check "Load preferences from a custom folder or URL"
3. Set the folder to: `~/Projects/dotfiles/iterm2`
4. Click "Save Current Settings to Folder"

## Restore Preferences

To restore iTerm2 settings on a new machine:

1. Open iTerm2 → Settings → General → Preferences
2. Check "Load preferences from a custom folder or URL"
3. Set the folder to: `~/Projects/dotfiles/iterm2`
4. Restart iTerm2

The settings are stored in `com.googlecode.iterm2.plist`.
